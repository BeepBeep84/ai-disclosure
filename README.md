# AI Transparency Levels

A simple, human-readable way to disclose how AI was used in the creation of writing, code, design, and other digital work.

## What this is

AI is becoming a normal part of writing, design, and development. It is a tool, not something to hide, my only request is for transparency.

This project suggests a simple disclosure label that anyone can use to show the level of AI involvement in a piece of work.

It is not a judgement of quality. It is only a transparency label.

## The format

Example label:

[ AI-2 | Created with AI collaboration ]

This label can be added to a website, README, project page, article, app, or other digital work, and should link to an explanation of the levels.

## The levels

### AI-1 - AI-assisted

AI was used as a support tool during the creation process.

Examples may include:
- spell checking
- proofreading
- debugging
- code review
- brainstorming
- help solving specific problems

The main writing, design, or development was still primarily done by a person.

Example label:

`[ AI-1 | AI-assisted ]`

## AI-2 - AI-collaborative

AI was used as an active partner in the process.

It helped produce meaningful parts of the final writing, design, or code, but a person directed, reviewed, edited, and shaped the final result.

Example label:

`[ AI-2 | Created with AI collaboration ]`

## AI-3 - AI-generated

Most or all of the final writing, design, or code was generated by AI.

A person may still have reviewed, selected, edited, or published the result, but the bulk of the output came from AI.

Example label:

`[ AI-3 | Mostly AI-generated ]`

## Why transparency matters

People may want to know whether something was:
- primarily human-made with light AI assistance
- created in collaboration with AI
- mostly generated by AI

This label exists to support clarity and trust.

It does not say whether a piece of work is good or bad.
It does not rank value.
It does not shame people for using AI.

It simply makes AI use visible.

## Suggested usage

Use the label as a link to this page, or to another page that explains the levels.

For example:

```md
[ AI-2 | Created with AI collaboration ](https://github.com/BeepBeep84/ai-disclosure/)
```
Or on a project page:
```md
This project is labelled [ AI-2 | Created with AI collaboration ](https://github.com/BeepBeep84/ai-disclosure/).
```
## Guidance

These levels are intentionally simple.

They are not meant to capture every edge case perfectly. They are meant to make disclosure easier and more normal.

If you are unsure which label fits best, choose the one that most honestly reflects how the final work was created.

## Goal

The goal is to encourage openness around AI use without stigma, fear, or moral panic.

AI is a tool. Transparency helps people use that tool honestly.
